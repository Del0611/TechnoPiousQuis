🐍 Python - Bahasa Pemrograman Serbaguna

Python adalah bahasa pemrograman tingkat tinggi yang dirancang oleh Guido van Rossum pada tahun 1991. Dikenal dengan sintaks yang bersih dan mudah dibaca.

Karakteristik Utama

  • Interpreted: Kode dieksekusi baris per baris
  • Dynamically Typed: Tipe data ditentukan saat runtime
  • Multi-paradigma: Mendukung OOP, prosedural, dan fungsional
  • Indentation-based: Menggunakan indentasi untuk blok kode

Contoh Program Dasar

# Program Hello World
print("Hello, Dunia!")

# Variabel dan tipe data
nama = "Ahmad"  # string
umur = 25  # integer
tinggi = 1.75  # float
is_student = True  # boolean

# Percabangan
if umur >= 18:
  print("Dewasa")
else:
  print("Di bawah umur")

Struktur Data

# List (array dinamis)
buah = ["apel", "jeruk", "mangga"]
buah.append("pisang")

# Dictionary (key-value)
siswa = {"nama": "Budi", "nilai": 85}

# Set (koleksi unik)
angka_unik = {1, 2, 3, 3, 2}  # hasil: {1, 2, 3}

# Tuple (immutable)
koordinat = (10, 20)

Fungsi & OOP

# Definisi fungsi
def sapa(nama):
  return f"Halo, {nama}!"

# Class (OOP)
class Mobil:
  def __init__(self, merk, tahun):
    self.merk = merk
    self.tahun = tahun

  def info(self):
    return f"{self.merk} ({self.tahun})"

mobil1 = Mobil("Toyota", 2023)
print(mobil1.info())
💡 Tahukah Kamu? Python adalah bahasa nomor 1 di dunia untuk AI dan Machine Learning karena ekosistem library-nya yang kaya (TensorFlow, PyTorch, scikit-learn, Pandas, NumPy).

💛 JavaScript - Bahasa Web Interaktif

JavaScript adalah bahasa pemrograman yang membuat halaman web menjadi interaktif. Dibuat oleh Brendan Eich pada tahun 1995, kini menjadi salah satu bahasa paling populer di dunia.

Karakteristik Utama

  • Client-side & Server-side: Berjalan di browser dan Node.js
  • Event-driven: Merespons aksi pengguna (klik, input, dll)
  • Prototype-based OOP: Pewarisan berbasis prototipe
  • Asynchronous: Mendukung async/await, Promise, callback
  • Weakly Typed: Tipe data fleksibel

Contoh Program Dasar

// Output ke console
console.log("Hello, Dunia!");

// Variabel (3 cara deklarasi)
let nama = "Ahmad";  // bisa diubah
const umur = 25;  // tidak bisa diubah
var tinggi = 1.75;  // cara lama (hindari)

// Arrow function (ES6+)
const sapa = (nama) => `Halo, ${nama}!`;

// Template literal
console.log(`Nama: ${nama}, Umur: ${umur}`);

Manipulasi DOM

// Mengakses elemen HTML
const judul = document.getElementById('judul');
const tombol = document.querySelector('.btn');

// Mengubah konten
judul.textContent = 'Judul Baru';

// Event listener
tombol.addEventListener('click', () => {
  alert('Tombol diklik!');
});

Asynchronous JavaScript

// Promise
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => resolve('Data berhasil diambil'), 1000);
  });
};

// Async/Await
async function getData() {
  const result = await fetchData();
  console.log(result);
}
💡 Tahukah Kamu? JavaScript adalah satu-satunya bahasa yang berjalan secara native di browser. Dengan Node.js, JavaScript kini bisa digunakan untuk backend, desktop, mobile, bahkan IoT!

🟠 HTML - HyperText Markup Language

HTML adalah bahasa markup standar untuk membuat struktur halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang menggunakan tag untuk mendefinisikan elemen halaman.

Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Heading 1</h1>
  <p>Paragraf teks.</p>
</body>
</html>

Elemen Penting HTML

TagFungsiContoh
<div>Kontainer umum (block)<div>...</div>
<span>Kontainer inline<span>teks</span>
<a>Hyperlink<a href="url">Klik</a>
<img>Gambar<img src="foto.jpg" alt="Deskripsi">
<form>Formulir input<form><input></form>
<table>Tabel data<table><tr><td></td></tr></table>
<ul>/<ol>List (unordered/ordered)<ul><li>Item</li></ul>

Formulir Input

<form action="/submit" method="POST">
  <label for="nama">Nama:</label>
  <input type="text" id="nama" name="nama" placeholder="Masukkan nama">

  <label for="email">Email:</label>
  <input type="email" id="email" required>

  <button type="submit">Kirim</button>
</form>

Semantic HTML5

<header>...</header>  ← Header halaman
<nav>...</nav>  ← Navigasi
<main>...</main>  ← Konten utama
<section>...</section>  ← Bagian konten
<article>...</article>  ← Artikel mandiri
<aside>...</aside>  ← Konten samping
<footer>...</footer>  ← Footer halaman
💡 Tips Penting Gunakan semantic HTML untuk meningkatkan SEO dan aksesibilitas. Search engine lebih memahami struktur halaman yang menggunakan tag semantic.

⚙️ C - Bahasa Pemrograman Prosedural

Bahasa C dikembangkan oleh Dennis Ritchie pada tahun 1972 di Bell Labs. C adalah fondasi dari banyak bahasa modern dan digunakan untuk sistem operasi, embedded systems, dan aplikasi performa tinggi.

Karakteristik Utama

  • Compiled: Kode dikompilasi menjadi machine code
  • Statically Typed: Tipe data harus dideklarasikan
  • Manual Memory Management: Menggunakan malloc/free
  • Pointer: Akses langsung ke alamat memori
  • Prosedural: Berbasis fungsi

Contoh Program Dasar

#include <stdio.h>

int main() {
  // Output ke console
  printf("Hello, Dunia!\n");

  // Variabel dengan tipe data
  int umur = 25;
  float tinggi = 1.75;
  char huruf = 'A';

  // Percabangan
  if (umur >= 18) {
    printf("Dewasa\n");
  } else {
    printf("Di bawah umur\n");
  }

  return 0;
}

Pointer dan Array

// Array
int angka[5] = {1, 2, 3, 4, 5};

// Pointer - menyimpan alamat memori
int x = 10;
int *ptr = &x;  // ptr menyimpan alamat x
printf("%d", *ptr);  // dereference: mengambil nilai

// Dynamic memory allocation
int *arr = (int*) malloc(5 * sizeof(int));
free(arr);  // selalu bebaskan memori!

Struct (Struktur Data)

// Definisi struct
struct Mahasiswa {
  char nama[50];
  int nim;
  float ipk;
};

// Penggunaan
struct Mahasiswa mhs1;
strcpy(mhs1.nama, "Budi");
mhs1.nim = 12345;
mhs1.ipk = 3.75;
⚠️ Perhatian! C tidak memiliki garbage collector. Setiap memori yang dialokasikan dengan malloc() harus dibebaskan dengan free() untuk mencegah memory leak.

🔵 C++ - Pengembangan C dengan OOP

C++ dikembangkan oleh Bjarne Stroustrup pada tahun 1985 sebagai pengembangan dari C dengan menambahkan fitur Object-Oriented Programming (OOP).

Karakteristik Utama

  • Multi-paradigma: OOP, prosedural, generik, fungsional
  • Compiled: Performa tinggi
  • STL: Standard Template Library yang kaya
  • RAII: Resource Acquisition Is Initialization
  • Smart Pointers: Manajemen memori otomatis

Contoh Program Dasar

#include <iostream>
using namespace std;

int main() {
  cout << "Hello, Dunia!" << endl;
  return 0;
}

Class & OOP

class Mobil {
private:
  string merk;
  int tahun;
public:
  Mobil(string m, int t) : merk(m), tahun(t) {}
  void info() {
    cout << merk << " (" << tahun << ")" << endl;
  }
};

Mobil mobil1("Toyota", 2023);
mobil1.info();

STL (Standard Template Library)

#include <vector>
#include <map>
#include <algorithm>

// Vector (array dinamis)
vector<int> angka = {1, 2, 3};
angka.push_back(4);

// Map (key-value)
map<string, int> nilai;
nilai["Budi"] = 85;

// Algorithm
sort(angka.begin(), angka.end());

☕ Java - Write Once, Run Anywhere

Java dikembangkan oleh James Gosling di Sun Microsystems (sekarang Oracle) pada tahun 1995. Slogannya "Write Once, Run Anywhere" karena Java berjalan di Java Virtual Machine (JVM).

Karakteristik Utama

  • Platform Independent: Berjalan di JVM
  • Strictly OOP: Semua kode dalam class
  • Statically Typed: Tipe data eksplisit
  • Garbage Collection: Manajemen memori otomatis
  • Multithreading: Dukungan konkurensi

Contoh Program Dasar

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, Dunia!");

    // Variabel
    int umur = 25;
    double tinggi = 1.75;
    String nama = "Ahmad";

    // Percabangan
    if (umur >= 18) {
      System.out.println("Dewasa");
    }
  }
}

OOP di Java

// Class dengan constructor
public class Mobil {
  private String merk;
  private int tahun;

  public Mobil(String merk, int tahun) {
    this.merk = merk;
    this.tahun = tahun;
  }

  public String getInfo() {
    return merk + " (" + tahun + ")";
  }
}

// Penggunaan
Mobil mobil1 = new Mobil("Toyota", 2023);
System.out.println(mobil1.getInfo());

ArrayList & Collections

import java.util.ArrayList;
import java.util.HashMap;

// ArrayList (array dinamis)
ArrayList<String> buah = new ArrayList<>();
buah.add("Apel");
buah.add("Jeruk");

// HashMap (key-value)
HashMap<String, Integer> nilai = new HashMap<>();
nilai.put("Budi", 85);

🟣 PHP - Hypertext Preprocessor

PHP dibuat oleh Rasmus Lerdorf pada tahun 1994. PHP adalah bahasa server-side yang dirancang khusus untuk pengembangan web dinamis.

Karakteristik Utama

  • Server-side: Dieksekusi di server, hasil dikirim ke browser
  • Embedded: Bisa disisipkan dalam HTML
  • Weakly Typed: Tipe data fleksibel
  • Database Integration: Built-in support untuk MySQL, PostgreSQL
  • Session Management: Cookie, session, form handling

Contoh Program Dasar

<?php
// Output
echo "Hello, Dunia!";

// Variabel (diawali $)
$nama = "Ahmad";
$umur = 25;
$tinggi = 1.75;

// Concatenation (.)
echo "Nama: " . $nama . ", Umur: " . $umur;

// Percabangan
if ($umur >= 18) {
  echo "Dewasa";
} else {
  echo "Di bawah umur";
}
?>

Form Handling (GET & POST)

<?php
// Menerima data dari form
if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $nama = $_POST["nama"];
  $email = $_POST["email"];
  echo "Halo, $nama! Email: $email";
}
?>

<form method="POST">
  <input type="text" name="nama">
  <input type="email" name="email">
  <button type="submit">Kirim</button>
</form>

Array di PHP

<?php
// Indexed array
$buah = array("Apel", "Jeruk", "Mangga");
// Atau cara singkat:
$buah = ["Apel", "Jeruk", "Mangga"];

// Associative array (key-value)
$siswa = [
  "nama" => "Budi",
  "nilai" => 85,
  "kelas" => "XII-A"
];

// Looping array
foreach ($buah as $item) {
  echo $item . "<br>";
}
?>
